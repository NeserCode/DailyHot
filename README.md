# Vue3Example
 a vue-cli example,contain Vue-Cli, ElementUI, TailWindCss, Axios and lots of things.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files
```
npm run lint
```
